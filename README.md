# Foodie

## A MERN app that allows the user to search for different meals or recipes, add ingredients into a grocery list and more!

A user is able to perform the following:

- Search for different recipes using the available APIs
- Create own food recipe
- Create a grocery list
- Create an account to save recipes and grocery lists

## APIs
- Spoonacular. [Link to Docs](https://spoonacular.com/food-api/docs)
- The Meal DB. [Link to Docs](https://www.themealdb.com/api.php)
